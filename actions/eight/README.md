# Eight-figure action

This action makes a robot to run eight-figure.

## Usage

Build YP-Spur program with Makefile.

```
make
```

Execute the pragram.

```
./eight
```

## Parameters

The inital parameters are:

  * `velocity`: 0.3
  * `accelaration`: 1.0
  * `angular_velocity`: 1.0
  * `angular_acceleration`: 1.0
  * `starting_point`: 0.0, 0.0, 0.0
